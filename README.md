Description
A simple calculator application designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The project was created to practice programming fundamentals, user interface design, and logical problem-solving.

The calculator provides a user-friendly interface where users can input numbers and instantly calculate results efficiently and accurately.

 Technologies Used
- Python 
- Android Studio / VS Code / NetBeans
- XML (if Android App)

Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear Button
- Simple User Interface
